*2025/12/31:好奇心が外発化されるとき*

「これやってみたい」「これってどうなってるんだろう」と思うのが研究のとっかかりだと思う．そうやってスタートした研究はどんどん手が進むし，やりたいことがでてくる．自ずと結果もついてくる．そうすると得られた結果を他の人に発表したいと思えるし，自分がどう考えたかを伝えたくなる．これが本来の研究発表なのかなって思う．でも，どこかのタイミングで「研究は進んでないのに発表はしなきゃいけない」なんて場面が来たりする．自分はそうだった．このとき「意味ある発表」が「義務の発表」に変わる．最初はいいかもしれない．でもこれが続くと，好奇心が評価のための義務作業として条件付けされる．これを好奇心の外発化と呼んでみる．自分はこれを経験したことで研究を楽しめなくなった．D1の冬だった．

まだ研究が楽しいと思えている大学院生に伝えたいのは，自分が発表したいと思っていないなら会議は参加しなくていいということ．たしかに会議に参加したことで本当に多くの人と知り合えた．これは財産．でも，そうやって参加した会議で自分の好奇心が壊れてしまったら，その人たちとのつながりも意味を失ってしまう．アカデミアに残るという選択肢をしなくなるのだから．こんなこと，自分は誰からも教えてもらえなかった．だからこそ，自分はここに書いておきたい．誰かのために．

*2025/12/22:ゲージ原理って?*

博論を書いていて「ゲージ原理」あるいは「ゲージ不変性」ってなんだろうと改めて考えた．テキトーにQFTの教科書を開いてみると，局所位相変換の下での理論の不変性，といった説明がされる．式のうえではそうだと思う．でもしっくりこない．九後さんの教科書（いわゆる九後ゲージ）では"場の値を測る座標軸の設定が時空の各点で独立に行えるべきという要請"と説明される．少し踏み込んではいるけれど，どう飲み込めばいいだろうか．

一つの考え方として，ゲージ原理は自然からの要求ではなく，人々が人為的に(by handに)導入した一種の構成原理とみなす，というのがあると思う．量子力学では波動関数の位相は（特殊な場合を除き）観測できない．その拡張として，位相因子=観測できないものを時空の各点でどう選んでも物理法則には影響しないでしょ，という立場．なにより，いったんゲージ不変性を受け入れてしまうと，私たちは自然をとてもうまく記述する理論を手にできる．標準模型がその最たる例である．

この最後の部分がまずいとも思う．つまり，受け入れた結果，うまくいっている，だからそういうもの，という論理．ゲージ原理に出会ってから8年ほどになると思うけれど，未だこういう論理に従って物理を知った気になっている自分への戒めとしてここに書き残しておきたい．
